## 1.0.0 - 2026-06-02

### Added
- Initial release of **availability_prerequisite**
- Restrict activity and section access until a student completes a prerequisite course
- Live search widget to quickly find courses by name on large Moodle sites
- Direct link to the prerequisite course shown in the restriction message
- Supports **must be complete** and **must not be complete** states
- Compatible with Moodle 4.1, 4.2, 4.3, 4.4, 4.5
- PHPUnit test suite
- Behat acceptance tests
- GDPR privacy provider — no personal data stored
- Backup and restore support
- Proper YUI module structure for Moodle core_availability framework
